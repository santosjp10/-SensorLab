📘 Sobre os índices

NDVI = (NIR - R) / (NIR + R) - Índice de Vegetação por Diferença Normalizada

SAVI = ((NIR - R) / (NIR + R + L)) * (1 + L) - Índice de Vegetação Ajustado ao Solo (L=0.5)

EVI = 2.5 * ((NIR - R) / (NIR + 6*R - 7.5*B + 1)) - Índice de Vegetação Melhorado

NDWI = (G - NIR) / (G + NIR) - Índice de Água por Diferença Normalizada

Os valores dos índices variam de -1 a 1. Vegetação saudável apresenta NDVI > 0.3.
